It's a game where you ship cargo by truck.😀 (This is an ongoing project, so there are some unfinished parts.)

About Key🔤

WASD: Drive truck 🚛
Space: Brake 🚧
N: On/Off Navigation 🧭


About UI 🖥

Navigation🧭: 
It is on the right bottom. 
If you turn on the navigation, the route to the target point is displayed.
Your location is 🔺

Speedometer🚴‍♂️:
It is on the left bottom.
The speed of the truck is displayed.
If the speed is less than 80, it is green🟢, if it is less than 140, it is yellow🟡, if it is 140 or more, it is red🔴.
